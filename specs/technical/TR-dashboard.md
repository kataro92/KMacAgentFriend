# TR-dashboard — Full panel (Phase 6)

## Purpose

Optional dashboard window: agent status, chat, tools, and knowledge placeholders.

## Requirements

**TR-DASH-001** Swift SHALL expose a dashboard window from the menu bar **Open Full Panel** action.

**TR-DASH-002** Dashboard SHALL show brain (connection/Ollama), chat (last turn), limbs (tool status), knowledge (placeholder).
